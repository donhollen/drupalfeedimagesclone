INTRODUCTION
------------

The Feeds Images Importer module is created with the only pourpos to achieve the 
missing functionality of Feed module which is using Feeds API, which so far can not get the images from RSS Feed URL location.


 * For a full description of the module, visit the sandbox project page:
   https://www.drupal.org/sandbox/kadiiskiffw/2704861

 * Or checkout the issue page:
   https://www.drupal.org/node/2704941

REQUIREMENTS
------------

This module requires the following modules:

 * Views (https://drupal.org/project/feeds)

INSTALLATION
------------
 
 * Install as you would normally install a contributed Drupal module. See:
   https://drupal.org/documentation/install/modules-themes/modules-7
   for further information.

CONFIGURATION
-------------
 
 * Configure user permissions in Administration » People » Permissions:

   - The module settings page requires "administer site configuration" permission.

   - Users with the allowed permissions will see the configuration page at:
     >>Configuration >>Content authoring >>Feeds Images Importer, or
     admin/config/content/feeds-images/settings
